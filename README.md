## Message Center

### Web3 messaging app with a sprinkle of Web2 for email & phone notifications.

#### Setup

Install dependencies

```
yarn
```

Populate `.env.local`

#### Local development

Run the oracle server:

```
yarn oracle-dev
```

Run the Base Inbox dashboard:

```
yarn dashboard-dev
```

Run the Sample App:

```
yarn sample-app-dev
```
